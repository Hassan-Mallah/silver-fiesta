# silver-fiesta
python + mongodb + FastAPI

### Setup

.env file example:

    MONGO_USERNAME=USERNAME
    MONGO_PASSWORD=PASSWORD

### Endpoints

- You can check endpoints here: {url}/docs
- Get post with post_id
- Get all posts
- Pagination for posts
- Update a post with post_id
- Delete a post with post_id
- Create a post
- Ping MongoDB :)
